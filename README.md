# Rust-I-learn

## How to Run

1. Open a terminal.
2. Navigate to the project directory:
    ```sh
    cd /home/mm/Desktop/Rust-I-learn
    ```
3. Compile and run the program using Cargo:
    ```sh
    cargo run
    ```

    If you are not using Cargo, you can compile and run your Rust program using `rustc`:
    ```sh
    rustc main.rs
    ./main
    ```

## Common Commands

- New project:
    ```sh
    cargo new <project_name>
    ```
- Build the project:
    ```sh
    cargo build
    ```

- Run the project:
    ```sh
    cargo run
    ```

- Test the project:
    ```sh
    cargo test
    ```

- Check the project for errors without building:
    ```sh
    cargo check
    ```

- Add a dependency:
    ```sh
    cargo add <dependency-name>
    ```

- Update dependencies:
    ```sh
    cargo update
    ```

- Clean the project (remove build artifacts):
    ```sh
    cargo clean
    ```
